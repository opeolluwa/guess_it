## Description

Classical guess the number game built in C programing language

## Getting Started

```shell
git clone https://github.com/opeolluwa/hestia
cd hestia 
mkdir build 
cmake .. 
```

### Dependencies
- [Cmake >=3.20](https://cmake.org)
- clang
### Installing
From the `./build` directory, run `make`
### Executing program
From the `./build` directory, run `./hestia`
